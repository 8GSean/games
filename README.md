# מרכז משחקי הרטרו (Retro Game Hub)

זהו מרכז משחקי רטרו פרימיום שנבנה במיוחד עבורך. כאן תמצא את הקלאסיקות הגדולות בעיצוב מודרני ומרהיב.

## המשחק הראשי:

### 🐢 צבי הנינג'ה: שומרי הביוב 3D
הרפתקת לגו תלת-מימדית ברחבי ניו יורק ומערכת הביוב — המשחק המוביל של המאגר.
- **דמויות**: לאונרדו (קטאנה), רפאל (סאי), דונטלו (בו) ומיכלאנג'לו (נונצ'אקו), כל אחד עם יכולת נינג'ה ייחודית.
- **מכניקה**: מצב שוטטות חופשית, איסוף מכלי מוטגן ופיצות (פיצה = בריאות!), קרב מול שרדר.
- **עיצוב**: מנוע לגו תלת-מימד (Three.js), עיר וביוב לחקירה, ותמיכה מלאה בשלט.
- **הפעלה**: [פתח את צבי הנינג'ה](ninja-turtles/ninja-turtles.html)

## המשחקים הנוספים:

### 1. 🐍 משחק סנייק (Snake) פרימיום
משחק הסנייק הקלאסי בעיצוב ניאון מודרני.
- **עיצוב**: Cyber-Neon, גבעול ועלה לתפוח, זוהר יוקרתי.
- **חדש**: בורר מהירות (קל, בינוני, קשה) ומערכת שיאים.
- **הפעלה**: [פתח את סנייק](snake/snake.html)

### 2. 🍒 משחק פקמן (Pac-Man) פרימיום
המשחק הקלאסי ביותר בגרסת ניאון כחולה מרשימה.
- **עיצוב**: קירות ניאון כחולים, נקודות כוח זוהרות, ורוחות צבעוניות.
- **מכניקה**: אכילת נקודות, בונוסים, ובריחה (או רדיפה) מהרוחות.
- **הפעלה**: [פתח את פקמן](pacman/pacman.html)

### 3. 🦾 לגו מרוול (LEGO Marvel) פרימיום 3D
משחק תלת-מימד מלא בסגנון ה-XBOX! בחר בין איירון מן לענק הירוק ועבור בעולם הלגו הפתוח.
- **דמויות**: איירון מן (מהיר) והענק הירוק (קפיצות ענק).
- **עיצוב**: תלת-מימד (Three.js), איסוף לבני לגו, וצללים בזמן אמת.
- **הפעלה**: [פתח את לגו מרוול](marvel-2/marvel-2.html)

### 4. 🦇 לגו באטמן 4 (LEGO Batman 4) פרימיום 3D
משחק לגו תלת-מימד מרהיב המבוסס על סדרת משחקי LEGO Batman המצליחה.
- **דמויות**: באטמן (בעל חליפות מתחלפות), רובין, באטגירל, נייטווינג, הג'וקר, הארלי קווין, איש החידות, ופלאש (The Flash) - אשר מהיר פי 7 משאר הדמויות ומשאיר שובל ברקים מרהיב מאחוריו!
- **באטמוביל**: קרא ונהג ברכב הבאטמוביל המפורסם, דרוס אויבים ונפץ חפצים ברחבי גות'אם.
- **בוס ענק**: קרב בוס מאתגר נגד ג'וקר-מך ענקי שיורה רקטות.
- **עיצוב וסאונד**: תלת-מימד (Three.js), אפקטים קוליים מסונתזים בזמן אמת (Web Audio API), וממשק משתמש של מחשב ה-Batcomputer.
- **הפעלה**: [פתח את לגו באטמן 4](lego-batman-4/lego-batman-4.html)

## איך משחקים?
- **סנייק/פקמן**: חצים או WASD.
- **לגו מרוול**:
    - **WASD**: תנועה.
    - **SPACE**: קפיצה (לקפוץ גבוה עם האלק!).
    - **SHIFT**: ריצה מהירה.
    - **כפתור "החלף דמות"**: מעבר בין איירון מן להאלק.
- **לגו באטמן 4**:
    - **WASD / מקשי חצים**: תנועה רגילה / נהיגה בבאטמוביל.
    - **SPACE**: קפיצה (החזק לחוץ לדאייה בחליפת תעופה או עם באטגירל).
    - **F / קליק שמאל / כפתור "יכולת"**: שימוש ביכולת הדמות או החליפה (זריקת באטראנג, ירי רקטות, מכת פטיש, פולס חשמלי, וירי כדורי אש בחליפת אש).
    - **C / כפתור "רכב"**: קריאה לבאטמוביל, כניסה או יציאה ממנה.
    - **E / Enter**: איפוס דמות או התאוששות מפגיעה.
    - **תפריט צד**: החלפת דמויות וחליפות (עבור באטמן, כולל חליפת אש בוערת חדשה!) בלחיצה קלה.
    - **🎮 תמיכה בשלט פלייסטיישן (PlayStation Controller)**:
        - **ג'ויסטיק שמאלי (Left Stick)**: תנועה במרחב / היגוי בבאטמוביל.
        - **כפתור X (Cross)**: קפיצה (החזק באוויר לדאייה).
        - **כפתור מרובע (Square)** / **L1 / R1**: שימוש ביכולת מיוחדת (יריית כדורי אש, טילים וכו').
        - **כפתור עיגול (Circle)**: זימון, כניסה ויציאה מהבאטמוביל.
        - **כפתור משולש (Triangle)**: החלפה מהירה בין דמויות הרוסטר.
        - **נסיעה בבאטמוביל**: האצה באמצעות **X** או **R2** (הדק ימני), רוורס ובלימה באמצעות **Square** או **L2** (הדק שמאלי).
